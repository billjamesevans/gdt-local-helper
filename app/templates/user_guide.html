User Guide

  1. Create a Project: give it a title, customer, revision, and units.
  2. Upload drawing PDFs: go to the project page → Drawings → upload. Only PDFs up to 25 MB.
  3. Add GD&T requirements: open Requirements, use the FCF Builder on the left. As you choose a symbol, tolerance, material condition, and datums, the preview updates. Click Add Requirement.
  4. Map annotations: open Annotate on a drawing, pick a requirement, select Box or Polygon, draw on the page, and Save Annotation.
  5. Click through: in View, click any highlighted region to open the linked requirement + plain-English explanation.
  6. Symbols palette: browse Symbols for meanings, examples, and tips.
  7. Insights: open Insights to see bonus tolerance at MMC/LMC, datum completeness, legacy control warnings, and more.
  8. Export: use /export/csv?project=<id> for CSV; use /export/report?project=<id> for a PDF summary.

Tip: This is a single-user, local-only tool. Avoid exposing it beyond 127.0.0.1.
